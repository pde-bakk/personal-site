# [Click to visit!](https://www.peerdb.nl)

## Build a development website

```shell
make up
```

#### To stop or down it

```shell
make stop ||
make down
```

## Build a production website

```
make up PROD=1
```

#### To stop or down it

```shell
make stop PROD=1 ||
make down PROD=1
```
